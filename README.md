 # 🐧 Holberton School – UNIX & Shell

This repository contains projects from the **Holberton School UNIX & Shell curriculum**, focused on mastering Linux command line, system navigation, scripting, and automation.

---

## 📌 Overview

This project builds strong foundations in:

- 🖥️ Linux / UNIX systems
- 📂 File system navigation
- ⚙️ Shell scripting (Bash)
- 🔐 Permissions & process management
- 🔎 System inspection & debugging

---

## 🧠 Key Concepts Learned

### 🖥️ Shell Basics
- Navigating directories (`cd`, `ls`, `pwd`)
- File operations (`cp`, `mv`, `rm`, `mkdir`)

### 🔐 Permissions
- Read, write, execute permissions
- Changing ownership (`chown`, `chmod`)

### 🔁 Redirections
- Input/Output redirection (`>`, `>>`, `<`)
- Pipes (`|`)
- Filters (`grep`, `wc`, `sort`)

### ⚙️ Variables & Expansions
- Environment variables
- Shell expansions
- Aliases

---

## ⚙️ Requirements

- **OS:** Ubuntu 20.04 LTS  
- **Shell:** Bash  
- All scripts must:
  - Start with `#!/bin/bash`
  - Be executable
  - Follow project style guidelines

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/holbertonschool-unix.git
cd holbertonschool-unix
```

Run scripts:

```bash
./script.sh
```
