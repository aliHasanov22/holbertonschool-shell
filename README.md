- 🖥️ Bash scripting fundamentals
- 🌍 Environment variables and shell expansions
- 🔁 Loops, conditions, and parsing
- ⚙️ Process inspection and control

## 🧠 What This Repository Covers

### 1) Init files, variables, and expansions
This section focuses on shell startup behavior, aliases, environment variables, local variables, arithmetic expansions, and number conversion.  
It builds comfort with how Bash stores and expands values during execution.

### 2) Loops, conditions, and parsing
This section introduces `for`, `while`, and `until` loops, conditional logic, and text parsing.  
It also includes practical tasks like FizzBuzz, reading structured files, and parsing Apache logs.

### 3) Processes and signals
This section covers process IDs, listing running processes, controlling jobs, and sending signals.  
It also moves into basic process supervision using PID files and management scripts.

## ⚙️ Requirements

- **OS:** Ubuntu / Linux
- **Shell:** Bash
- Scripts should typically:
  - start with `#!/usr/bin/env bash` or `#!/bin/bash`
  - be executable
  - follow Holberton project requirements

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/holbertonschool-shell.git
cd holbertonschool-shell
```

Make a script executable if needed:

```bash
chmod +x filename
```

Run a script:

```bash
./filename
```

Example:

```bash
./init_files_variables_and_expansions/1-hello_you
./loops_conditions_and_parsing/10-fizzbuzz
./processes_and_signals/0-what-is-my-pid
```
